## proxmox
- 这是一个以操作系统的形式存在的虚拟机平台
- 管理员可以通过访问网页来管理这些虚拟机

## 安装
- 首先需要去官网下载一个合适的镜像
- https://www.proxmox.com/en/downloads/proxmox-mail-gateway/iso
